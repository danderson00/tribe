Global Options

Global options can be set on the TC.options object.

TC.options.basePath = 'Panes';
Name Type Default Description
basePath String Root path to load panes from
synchronous Boolean false Load resources and execute message subscribers synchronously
splitScripts Boolean false Split loaded scripts on each sourceURL tag and execute individually
handleExceptions Boolean true Handle exceptions within the framework
loadStrategy String adhoc Name of the registered load strategy to use
events [String] See reference Array of ordered event names to execute in the pane rendering pipeline
