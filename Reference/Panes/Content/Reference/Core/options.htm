Global Options

TC.options = {
    basePath: '',
    synchronous: false,
    splitScripts: false,
    handleExceptions: true,
    loadStrategy: 'adhoc',
    events: ['loadResources', 'createPubSub', 'createModel', 'initialiseModel', 
        'renderPane', 'renderComplete', 'active', 'dispose']
}
  • basePath - root path to load panes from
  • synchronous - perform load operations synchronously
  • splitScripts - split any loaded scripts on sourceURL tags and load individually
  • handleExceptions - handle exceptions raised - set to false in debug mode
  • loadStrategy - string containing the name of the registered load strategy to use
  • events - array of ordered event names to execute in the pane rendering pipeline
