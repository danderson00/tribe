Creating Panes

Panes can be created using the pane binding handler or with JavaScript using the core API functions.

<div data-bind="pane: paneOptions"></div>

paneOptions can either be a string representing the path to the pane, or an object with the following properties

{
    path: string,
    data: any,
    transition: string,
    reverseTransitionIn: boolean,
    handlesNavigation: string | navigationOptions,
    id: any,
    skipPath: boolean
}
