MessageHub Server Configuration API

Configuration must start with ConfigureHub.With() followed by a container configuration. Currently only Unity is supported. 

using Tribe.MessageHub.Core.Configuration;
using Tribe.MessageHub.Containers.Unity;

ConfigureHub.With().Unity(container).StartHub();

The following extension methods are then provided, all chainable:

using Tribe.MessageHub.Core.Configuration;
using Tribe.MessageHub.Containers.Unity;
using Tribe.MessageHub.ChannelPersisters.SqlServer;
using Tribe.MessageHub.Buses.NServiceBus;

ConfigureHub.With()
    .Unity(container)
    .ChannelAuthoriser<MyChannelAuthoriser>()
    .SqlServerPersistence("Data Source=.;Initial Catalog=Tribe.Channels;Integrated Security=true")
    .NServiceBus(bus)
    .MessagesFrom(typeof(ExampleMessage).Assembly)
    .StartHub();
