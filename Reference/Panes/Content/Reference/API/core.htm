Core API
TC.run = function (resourcesToPreload, initialModel)

Start Tribe.Composite, ensuring the specified resources are loaded first.

  • resourcesToPreload - array of strings containing URLs to required HTML, CSS or JS resources
  • initialModel - object to pass as the model to the initial ko.applyBindings call
TC.createNode = function (element, paneOptions, parentNode, context)

createNode creates a new Pane object and binds it to the specified element with the specified pane options.

  • element - selector | HTMLElement | Node | Pane
  • paneOptions - object containing pane options
  • parentNode - Node to be used as the parent node
  • context - Context object to be used

TC.registerModel = function (modelConstructor, options, resourcePath)
