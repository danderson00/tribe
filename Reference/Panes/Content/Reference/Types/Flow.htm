Flow Helpers

The Flow object exposes a number of functions to simplify handler definitions.

Flow Definitions

Flows are defined by creating a constructor function:

function MyNavigationFlow(flow, args, ...) { }

The object can expose the following properties:

function MyNavigationFlow(flow, argument1, argument2) {
    this.handles = {
        onstart: flow.to('startPane'),
        'messageTopic': flow.to('anotherPane', argument1),
        'startChildTopic': {
            'customHandler': function(messageData) {
                flow.navigate('anotherPane', messageData.property);
            }
        },
        'endFlowTopic': flow.endsAt('endPane')
    };
        
    this.endsChildrenExplicitly = true;
}
