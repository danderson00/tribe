Including and Excluding Sets of Files

The include and exclude options are available when using the pack, sync or zip commands. Files can be specified as simple string specifications, as an object or an array of both. Options available are:

Options such as recursive and template override the values set in the main configuration.

Examples

A simple string specification.

include: 'Scripts/*.js'

As an object that specifies additional options.

include: { files: 'Templates/*.htm', template: 'embedTemplate' }

A more complex example.

pack({
    to: 'Build/site.js',
    include: [
        { 
            files: 'Scripts/*.js', 
            template: { name: 'sourceUrl', data: { prefix: '/Source/' } },
            first: 'intro.js',
            last: 'outro.js',
            recursive: false
        }, {
        { 
            files: 'Templates/*.htm', 
            template: 'embedTemplate' 
        }, {
            files: 'Styles/*.css',
            template: 'embedCss'
        }
    ],
    exclude: '*.debug.js',
    outputTemplate: 'license',
    recursive: true,
    minify: true
});
