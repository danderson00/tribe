pack(options);

The pack function is used to combine and transform sets of files into a single output. The following options can be used:

Examples

pack({
    to: 'Build/site.min.js',
    include: '*.js',
    exclude: 'debug.js',
    minify: true
});

This example combines all of the JavaScript files in the same directory as the configuration file, minifies the output and writes it to site.min.js in the Build folder. 

pack({
    to: 'Build/site.js',
    include: [
        'Scripts/*.js',
        { files: 'Templates/*.htm', template: 'embedTemplate' },
        { files: 'Styles/*.css', template: 'embedCss' }
    ]
});

This example combines JavaScript files in the Scripts folder, HTML templates from the Templates folder and CSS files from the Styles folder into a single file. The templates and stylesheets are embedded into the output JavaScript using the built in templates embedTemplate and embedCss.
