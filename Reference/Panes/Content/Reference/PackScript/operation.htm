PackScript

Operation

PackScript is distributed as a Windows console application. A node.js version will be released in the near future. PackScript binaries can be obtained from github and are included in the Tribe download.

PackScript.exe has the following command line syntax: 

PackScript.exe ["Directory\To\Process\"] [/watch]

If no directory is specified, the current working directory is used.

/watch causes PackScript to remain active and keep all output files up to date as files change. The console can also be used to interrogate or execute JavaScript members.

Configuration Files

PackScript scans recursively through the target directory structure and finds all files fitting the following specifications:

*pack.config.js JavaScript configuration files. These are executed before any other file is processed.
*pack.js JavaScript configuration files.
*.template.* These files are loaded as templates and can be accessed by name using the template option.
