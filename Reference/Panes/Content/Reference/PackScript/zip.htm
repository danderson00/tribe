zip(options[, options, ...])

The zip function is used to create a compressed file in ZIP format. The following options can be used:

Similar to the pack function, the sync function returns an object containing a function called 'to'.

Examples

A simple flat ZIP file.

zip('Build/*.*').to('Build.zip');

This compresses the entire directory structure underneath the Build directory to Content/Build.zip.

zip({
    to: 'Content/Build.zip',
    include: 'Build/*.*',
    recursive: true
});
