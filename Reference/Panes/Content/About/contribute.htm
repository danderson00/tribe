Contribute

Tribe needs you!

There are many cool things on the roadmap for the Tribe platform like mobile device skins for each platform, seamless scale-out of the MessageHub component, better distributed event sourcing support, Azure Mobile Services integration and much more.

If you'd like to get involved, feel free to fork the github repository and submit pull requests. If you'd like to be part of the project, please email tribejs@gmail.com, tweet @tribejs or use the feedback button at the top of the page.
