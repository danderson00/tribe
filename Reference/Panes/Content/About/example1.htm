If you're not familiar with knockout, we recommend you have a quick run through the excellent tutorials at http://learn.knockoutjs.com/.

Tribe allows you to easily break your UI down into "panes" that can consist of a JavaScript model, HTML template and CSS stylesheet.

Simply create these files and refer to the pane by name. No complex configuration is required.

Tribe gives you an awesome debugging experience. Using the developer tools built in to Google Chrome, your file structure is preserved and each individual JavaScript file is fully debuggable.

To see this in action, head over to the debug version of the site using Chrome and check out the "Sources" tab in the developer tools.

For more practical examples, check out the webmail tutorial, or the source to this site.
