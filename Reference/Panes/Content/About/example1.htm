Show Me!

Tribe allows you to easily break your UI down into "panes" that can consist of a JavaScript model, HTML template and CSS stylesheet.

Simply create these files and refer to the pane by name. No complex configuration required.

This is pretty trivial example. You can check out the source to this site or the samples.

If you're using Chrome, head over to the debug version of the site and look at the source in the developer tools.
