Show Me!

Tribe allows you to easily break your UI down into "panes" and communicate between these panes using a publish / subscribe message bus.

Each pane can consist of a JavaScript model, a HTML template and CSS stylesheet. Simply create these files and refer to the pane by name.

Tribe does the rest - loads the resources, renders the template and binds an instance of the model to it.

This is, of course, pretty trivial. Check out the samples or the source to this site
